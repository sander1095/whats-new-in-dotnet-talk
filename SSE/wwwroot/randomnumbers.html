üìù How It Works

1. Server-Side (ASP.NET Core 10)

// Simple SSE endpoint app.MapGet("/api/randomnumbers", (CancellationToken ct) => { return TypedResults.ServerSentEvents(GenerateRandomNumbers(ct)); }); static async IAsyncEnumerable<int> GenerateRandomNumbers( [EnumeratorCancellation] CancellationToken cancellationToken) { var random = new Random(); while (!cancellationToken.IsCancellationRequested) { yield return random.Next(1, 1000); await Task.Delay(1000, cancellationToken); } }

2. Client-Side (JavaScript)

// Connect to SSE endpoint const eventSource = new EventSource('/api/randomnumbers'); // Listen for messages eventSource.onmessage = function(event) { const number = event.data; document.getElementById('numberDisplay').textContent = number; }; // Handle connection events eventSource.onopen = () => console.log('Connected'); eventSource.onerror = () => console.log('Connection error');
